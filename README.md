# Chat_Web_App

### For development use, not production (simplifying development workflow)

## Setup

Ensure you have python 3.6\8+ installed.

```bash
pip install -r requirements.txt
```

## Running the Server

```bash
python server.py
```

Flask_cors used because of Werkzeug limitation issues.
